Guess my number

Game instructions

  1. The game involves guessing a mysterious number using the fewest possible attempts.
  2. Choose a number from 1 to 25 and enter it in the empty field on the left.
  3. If the guessed number is too large or too small, a message will appear instructing you to choose another number.
  4. Upon correctly guessing the mysterious number, a winning message will appear. If the result is the best so far, it will be recorded in the "highscore" field.
  5. To reset the attempt, press "Try again."
  6. Have fun! 😀
